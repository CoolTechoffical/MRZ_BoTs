Mangandi API Installation Guide

Step 1: Install Mangandi API

pip install Mangandi

Add the above command to your terminal to install the Mangandi API. Ensure you have Python and pip installed on your system. This will install the Mangandi package required for your project.

Step 2: Add to Your Repository

Mangandi

Add the above line to your `requirements.txt` file to ensure that the Mangandi API is included when you deploy or share your project. This will allow others to install the required package easily.
